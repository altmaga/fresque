Faire Corps - Adrien M & Claire B

Faire de l’art avec du code

L’atelier, concu en collaboration avec la compagnie Adrien M & Claire B, propose d’appréhender une autre finalité singulière du code informatique, celle de créer des programmes à visée esthétique et visuelle.

Les participant·e·s testent le langage Java à travers l’application Code Combat. Il·elle·s découvrent des créations développées avec Processing et se familiarisent avec l’idée de code caché sous les coulisses d’un programme.
